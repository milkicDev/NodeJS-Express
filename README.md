## Features

- MongoDB

## Installation

- `git clone git@github.com:milkicDev/NodeJS-Express.git`
- `cd NodeJS-Express`
- `npm install`
- [Install & Start MongoDB](https://docs.mongodb.com/manual/tutorial/getting-started/)
- `npm start`

## Routes

- [localhost:3000](http://localhost:3000)
  - `/users`
    - `/ID`
    - `/create`
